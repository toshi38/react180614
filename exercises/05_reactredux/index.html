Start by copying the Redux parts of your app from Step 3 or 4 of exercise 4 into the Starter. If you didn't get to Step 3 or 4 (or if you're not happy with your solution) then you can simply copy the solution from step 4.

Once you've copied in the code you'll need to:

  • Create the React UI
  • Combine them using ReactRedux
